This tool is used to generate GML template and instance files for SNAP road network graph

USAGE: GMLGenerator -input <dataset/file/name> -directed <directed> -instances <no/of/instances> -out <output/directory>

<dataset/file/name> - File path of input graph
<directed> - Specify true, if the graph is directed
<no/of/instances> - Specify the number of instances to be generated
<output/directory> - Specify the directory to generate GML files

Supported vertex properties are:
license_plate
vehicle_type
vehicle_color
license_state
intersection_id

Supported edge properties are:
speed_limit
distance
